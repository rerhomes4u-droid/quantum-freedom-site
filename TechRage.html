Phone/Computer Storage Full

"Cannot take photo" or "Cannot install update"

  • iOS: Settings ‚Üí General ‚Üí iPhone Storage ‚Üí Review Large Attachments
  • Android: Files by Google app ‚Üí Clean ‚Üí Delete junk files
  • Windows: Disk Cleanup (type in start menu) ‚Üí Clean system files
  • Mac: About This Mac ‚Üí Storage ‚Üí Manage ‚Üí Review files
Secret Hack: Clear cache on streaming apps (Netflix, Spotify) - they store GBs of temporary data!

Device Running Super Slow

Lagging, freezing, taking forever

  • Universal: Restart it. Seriously. The oldest trick still works 90% of time.
  • Computer: Task Manager (Ctrl+Shift+Esc) ‚Üí End high-usage tasks
  • Phone: Clear RAM: On iPhone, quickly press volume up, volume down, then hold power until Apple logo
  • Browser: Extensions ‚Üí Disable/remove ones you don't use
Secret Hack: Disable animations: Windows "Adjust appearance" or Android Developer Options ‚Üí Animation scale 0.5x

Update Failed / Bricked Device

Stuck in update loop or worse after update

  • iOS Update Loop: Force restart (varies by model) ‚Üí Connect to iTunes/Finder
  • Windows Won't Update: Windows Update Troubleshooter (search in Start)
  • Mac Update Stuck: Safe Mode (Shift on boot) ‚Üí Try update again
  • Android Bricked: Recovery Mode (Power+Volume Down) ‚Üí Wipe cache partition
Secret Hack: Update via Ethernet cable instead of WiFi - much more stable connection for large updates

Password / 2FA Locked Out

Can't remember password or lost 2FA device

  • Apple ID: iforgot.apple.com ‚Üí Account recovery (takes days but works)
  • Google Account: account.google.com/signin/recovery ‚Üí Answer questions
  • Lost 2FA: Look for backup codes (check email for "backup codes")
  • Prevention: Use password manager (Bitwarden = free), print backup codes
Secret Hack: Use security questions with fake answers only you know (e.g., "Mother's maiden name" = "Superman")

WiFi / Internet Problems

Connected but no internet or slow speeds

  • First: Unplug router/modem, wait 30 seconds, plug back in
  • Windows: Command Prompt as admin ‚Üí "netsh winsock reset"
  • DNS Fix: Change DNS to 8.8.8.8 and 8.8.4.4 (Google) or 1.1.1.1 (Cloudflare)
  • Channel Conflict: Use WiFi analyzer app to find least crowded channel
Secret Hack: Aluminum foil behind router = DIY signal reflector for better direction
